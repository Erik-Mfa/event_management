Help - Event Messaging App

Overview

The Event Messaging App allows users to create events and send messages associated with those events. Users can submit event details through a simple form, and the events will be listed for everyone to view.

How to Use the App

Creating an Event

To create an event:

  1. Enter the Event Name in the provided text field.
  2. Provide your User ID (required for identification).
  3. Type your message in the Message textarea.
  4. Click on the Create Event button to submit your event.

After submission, the event will be added to the event list displayed on the same page.

Viewing Events

All created events are displayed in the Event List section below the form. Each event shows:

  • Event Name
  • Date
  • Message
  • User ID

Technical Details

The application uses PHP and AJAX for seamless interaction:

  • Form submissions are handled via AJAX, allowing the page to update without a full refresh.
  • Events are fetched from the server using AJAX, ensuring the latest events are always displayed.
